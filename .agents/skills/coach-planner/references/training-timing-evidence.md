# Training Timing — Evidence-Based Reference

排期时段决策的循证依据。基于 2024–2025 最新 meta-analyses 和 RCT。

## 核心结论 (TL;DR)

1. **训练时段对增肌/力量增益无显著影响** — 一致性 (consistency) 远比时段重要
2. **真正的变量是"训练结束到入睡的间隔"** — 不是早晚本身
3. **晚间阻力训练 ≠ 晚间耐力训练** — 阻力训练对睡眠的负面影响远小于耐力
4. **个体节律 (chronotype) 调节作用显著** — 早型人晚训扰睡更明显

## 关键 Evidence

### 1. Leota et al. 2025 (Nature Communications) — 史上最大样本

**样本**: 14,689 WHOOP 用户 × 1 年生物监测数据，dose-response 分析

**Dose-response 结果**:

| 训练结束距离睡眠 | 睡眠影响 | 推荐 |
|---|---|---|
| **≥ 4h** | 与不练日无差异（任何强度都安全） | ✅ 最优 |
| **2h** | 轻度运动 -15min / 高强度 -45min 睡眠 | ⚠️ 可接受 |
| **< 1h** | 显著紊乱：入睡延迟 / 时长↓ / 次日 RHR↑ HRV↓ | ❌ 避免 |

**机制**: 晚训降低副交感活性 → RHR 升高 + HRV 下降直到次日

### 2. 阻力 vs 耐力 (Frontiers Physiology 2024)

晚间不同训练类型对睡眠的差异化影响:

| 指标 | 阻力训练 | 耐力训练 |
|------|---------|---------|
| Sleep efficiency | ↓ 2-3% | ↓ 2-3% |
| **Cortisol 升高** | 较低 | **显著更高** (p=0.038) |
| **Beta arousal (N2 睡眠)** | **↓ -0.046**（实际利好）| ↑ +0.025（紊乱）|
| Total sleep time | 不显著 | ↓ 趋势 |

**关键发现**: 晚做**阻力训练对睡眠的负面影响远小于耐力训练**。重训反而降低 N2 期 beta arousal — 可能助眠。

**实操含义**: Z2 / cardio / 跑步 必须 AM 或周末，**不要塞晚上**。

### 3. 时段 vs 增肌 (2025 ACSM Position Stand + Sports Medicine Open)

- **力量增益不受时间影响** (ACSM 2025)
- **14 个 meta-analyses + 178 研究**: 时间对肌肥大幅度无直接影响 (Sports Medicine Open)
- **2025 系统综述 (14 研究 / 716 人)**: morning vs evening 在体成分 / BMI / 睡眠延迟 / 血脂上无显著差异

**结论**: 增肌只看 volume / frequency / intensity / progression，**不看时段**。

### 4. Chronotype 调节

- **早型人** (Early types): 晚训 → 增加觉醒次数 + insomnia 风险
- **晚型人** (Late types): 晚训甚至可能改善睡眠
- 个体差异需要 1-2 周试验确认

## 排期决策树

排期 coach 应用:

```
用户家里到家时间 X，目标灯灭时间 Y
→ 计算最早可用训练结束时间 = X + 15min（换衣/准备）+ 训练时长
→ 比对距灯灭间隔:
   - ≥ 4h → 任何强度都 OK
   - 2-4h → 阻力 OK, 高强度耐力 NOT OK
   - 1-2h → 仅低强度阻力 OK
   - < 1h → 推迟训练 OR 推迟灯灭 OR 改晨练
```

**示例**:
- 19:30 到家 + 22:00 灯灭 → 75min 训练 21:00 结束 = **1h 间隔（红线区）**
- 解决方案 (按 EV 排序):
  1. 改晨练 06:00-07:15 (距前晚睡 7h+，距当晚睡 14h，零干扰)
  2. 推迟灯灭到 23:00 (2h 间隔，可接受区)
  3. 缩短训练到 45min (20:30 结束 → 1.5h 间隔)
  4. ❌ 22:00 灯灭硬上：长期 HRV 趋势会被压低
